# Student Registration System (Syntecxhub Internship Task)

## Description
This project is a console-based Student Registration System developed using Java and MySQL. It allows users to manage student records efficiently.

## Features
- Add Student
- View Students
- Update Student
- Delete Student

## Technologies Used
- Java
- MySQL
- JDBC

## Concepts Used
- CRUD Operations
- Database Connectivity (JDBC)
- SQL Queries (INSERT, SELECT, UPDATE, DELETE)

## How to Run
1. Create database in MySQL:
   CREATE DATABASE student_db;

2. Compile the program:
   javac StudentManagement.java

3. Run the program:
   java StudentManagement

## Output
Menu-driven system to manage student records using database.
